## 1. Led Blink
Far lampeggiare un LED a intervalli regolari.

**Materiali:**
- 1x Arduino UNO
- 1x LED
- 1x Resistenza 220
- Breadboard e cavi

**File:**
- `1-ledblink.png` - Schema del circuito
- `1-ledblinkbozzolini.ino` - Codice sorgente

## 2. Semaforo
Simulazione di un semaforo con tre LED (rosso, giallo, verde) che si alternano automaticamente.

**Materiali:**
- 1x Arduino UNO
- 1x LED rosso
- 1x LED giallo
- 1x LED verde
- 3x Resistenza 220
- Breadboard e cavi

**File:**
- `2-semaforo.png` - Schema del circuito
- `2-semaforonbozzolini.ino` - Codice sorgente

## 3. Semaforo con Pulsante
Semaforo che cambia stato alla pressione di un pulsante.

**Materiali:**
- 1x Arduino UNO
- 1x LED rosso
- 1x LED giallo
- 1x LED verde
- 3x Resistenza 220
- 1x Pulsante (push button)
- 1x Resistenza 10k (pull-down)
- Breadboard e cavi

**File:**
- `3-semaforopulsante.png` - Schema del circuito
- `3-semaforopulsantebozzolini.ino` - Codice sorgente

## 4. Incrocio
Simulazione di un incrocio con 4 semafori sincronizzati.
I semafori opposti si comportano allo stesso modo, alternandosi con quelli perpendicolari.

**Materiali:**
- 1x Arduino UNO
- 4x LED rosso
- 4x LED giallo
- 4x LED verde
- 12x Resistenza 220
- Breadboard e cavi

**File:**
- `4-semaforo2-2.png` - Schema del circuito
- `4-semaforo2-2Bozzolini.ino` - Codice sorgente
